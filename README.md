# GEN-AI Marketing Content Generator

## Overview
This project is a GenAI-based Marketing Content Generator that creates professional marketing content using Prompt Engineering techniques.

## Features
- Generate marketing content from basic topics
- Structured prompt engineering
- Data storage using JSON/Database
- Modular Python architecture

## Tech Stack
- Python
- OpenAI API
- Prompt Engineering
- JSON Database

## Project Structure
- app.py – Main application
- content_generator.py – Content generation logic
- prompt_engineer.py – Prompt design
- database.py – Data handling
- marketing_data.json – Sample data
